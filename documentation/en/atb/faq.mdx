---

### How is ATB data collected?
The ATB service transmits firewall log metadata directly to our servers via API.

### Where is the collected data stored?
Aegister stores the data in a Postgres database on GCP, hosted on servers in Milan.

### What is the data retention period?
ATB keeps only the latest 100k events. Older data is overwritten.

### How is the data protected during transmission and storage?
TLS 1.2 secures the data in transit. Authentication systems protect it at rest.

### Who has access to the data collected by ATB?
Only system administrators and designated system accounts can access it.

### How is the data deleted?
New events overwrite old ones, so ATB retains only the most recent 100k logs.
