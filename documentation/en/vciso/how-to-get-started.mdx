This guide explains how to start using the services in the Aegister Virtual CISO plan. It covers how to create or access your account, verify your license, and navigate the "Cyber Assurance" section.

## 1. Create or Access Your Account

To get started:

- **Create an account**: If you don't have one, follow the [account creation and first login](/documentation/en/acc/how-get-account) guide.
- **Log in**: If you already have an account, sign in at [app.aegister.com](https://app.aegister.com).

![ACC first login](/images/acc-dashboard-no-licences.png)

## 2. Verify Your Virtual CISO License

After logging in, check that the Virtual CISO license is active for your [Organization](https://app.aegister.com/organizations/). You need an active license to access the Virtual CISO services.  
![ACC organizations](/images/acc-organizations.png)  
To obtain a license, contact us.  
![ACC organization no license](/images/acc-organization-no-license.png)

## 3. Navigating the "Cyber Assurance" Section

Once your license is confirmed, go to the **Cyber Assurance** section. It offers two main services:
- **Cyber Posture**: shows the current status of your security policy implementation and compliance standards. View it [here](https://app.aegister.com/cyber-assurance/).  
  ![Cyber Assurance - Cyber Posture](/images/acc-cyber-posture.png)
- **Association**: lets you add new security frameworks and complete the required tasks. Learn more in the [Cyber Association](https://app.aegister.com/cyber-assurance/associations/) section.  
  ![Cyber Assurance - Associations](/images/acc-associations.png)

## 4. Structure of the Cyber Assurance Section

The **Cyber Assurance** section has two subsections:

### 4.1 Cyber Posture
- **Audience**: all users within an organization.
- **Functionality**: shows the implementation status of the security activities defined in the frameworks associated with your organization.

### 4.2 Association
- **Audience**: security officers and owners.
- **Functionality**:
  - Shows the security frameworks that Aegister has added to your organization.
  - Lets you add new security frameworks yourself (Aegister tailors them to your needs).  
    ![Associations - Security Framework](/images/acc-new-association.png)
  - Starts the required remediation tasks.

## 5. Task Management

Tasks fall into two types:

- **Automated**: scheduled scripts defined in the security framework that run periodically, with a log of past executions.
- **Self**: tasks that require manual work.  
  ![Associations - Self View](/images/acc-associations-self-view.png)

### Task Evaluation and Documentation

- **Evaluate**: click "evaluate" within a framework to start answering the assessment questions.  
  ![Associations - Evaluate](/images/acc-associations-evaluate.png)
- **Attachments and completion**: attach files as evidence, or click "completed" to mark a task as finished.
- **Versioning**: in the Cyber Posture section, view the version history of responses and attachments to track your progress.  
  ![Cyber Posture - Versioning](/images/acc-cyber-posture-versioning.png)

These steps let you start using the Aegister Virtual CISO plan and manage your organization's security tasks.
