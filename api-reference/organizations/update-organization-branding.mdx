description: "Update an organization's branding.

Accepts:
- palette: one of 'blue', 'purple', 'teal', 'green', 'orange', 'red', 'indigo', 'pink', 'slate', 'custom'
- custom_colors: an object with primary, primary_hover, and primary_dark hex colors

Only super_admin users can update branding."
openapi: put /api/v1/organizations/{id}/branding
