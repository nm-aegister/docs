---
title: Delete Specific Workflow Evaluations From An Organization
sidebarTitle: Delete Workflow Evaluations
openapi: delete /api/v1/organizations/{organizationid}/workflows/{workflowid}/evaluations/
---

Permanently deletes all workflow evaluations for the specified organization and workflow.

**Path params**
- `organizationid` — Organization ID
- `workflowid` — Workflow ID
