title: "Associa un workflow a un'organizzazione"
sidebarTitle: "Associa workflow"
description: "Abilita un framework di conformità (workflow) per un'organizzazione, ad esempio associa NIS 2 o ISO 27001:2023. È ciò che fa l'azione «aggiungi framework» della GUI: dopo l'associazione, controlli, attività e documenti del framework diventano disponibili per la valutazione. Ottieni l'id del workflow dall'elenco dei workflow disponibili."
openapi: post /api/v1/v-ciso/organizations/{organizationid}/workflows/{workflowid}
