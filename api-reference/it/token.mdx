## Come creare un token

Crea un token di autenticazione in quattro passaggi:

1. **Apri il tuo profilo**  
   Clicca sull'icona del profilo in alto a destra nella pagina, oppure vai direttamente su [Impostazioni → Profilo](https://app.aegister.com/settings/profile/).

2. **Crea un nuovo token**  
   Nella sezione Token API, clicca su *+ Nuovo*.
   ![Passaggio 2](/images/token1.png)

3. **Assegna un nome al token**  
   Inserisci un nome e clicca su "Genera".
   ![Passaggio 3](/images/token2.png)

4. **Copia il token**  
   Copia subito il token. Non verrà più mostrato.
   ![Passaggio 4](/images/token3.png)


Conserva il token in modo sicuro. Ogni richiesta API lo richiede.
